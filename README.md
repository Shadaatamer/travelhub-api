# TravelHub

A full-stack tour booking application built with Node.js, Express, MongoDB, Pug, and Stripe.

## Live Demo
https://travelhub-api.vercel.app

## Features
- User signup, login, logout
- JWT authentication with secure cookies
- Tour listing and tour details
- User account settings
- Password update
- Reviews system
- Stripe checkout integration
- Responsive Pug templates
- MongoDB/Mongoose data models
- Security middleware: rate limiting, sanitization, HPP, Helmet

## Tech Stack
Node.js, Express, MongoDB, Mongoose, Pug, Axios, Stripe, Vercel

## Environment Variables
This project requires environment variables such as DATABASE, JWT_SECRET, STRIPE_SECRET_KEY, and email credentials.
